# Brand Analysis System

**One line:** Her "@adrianavmarquez brand brain" — a structured market-research +
creative-intelligence engine that feeds every content decision from data.

This is a **hub (MOC) in Notion**, not a single doc. It's the operating system she
(and clients) use to go from raw research → messaging → creative pipeline. Preserved
here as a map so Claude knows the system exists and where each part lives.

## The modules
| Module | What it holds |
|---|---|
| **Basecamp** | Important internal links / home |
| **The Product** | Product details & comparison |
| **The Customer** | Customer data & personas |
| **Performance Audit** | High-level creative insights |
| **Brand Mentions** | Press & review sites (Weekly Brand Pulse) |
| **Reviews** | Customer review analysis |
| **Us VS Them** | Competitor analysis |
| **Features / Benefits** | Key language about the product |
| **Messaging and Angles** | Key language about the customer |
| **Social Media Portal** | Analysis of organic content |
| **Market Research Dashboard** | The system all data is extracted from |
| **Creative Roadmap** | The creative & testing pipeline |
| **Creative Intelligence Feed** | Ongoing signal |
| **Key Terms & Definitions** | Shared vocabulary across sections |
| **Brand Insight** | Internal analysis for performance & scale |
| **Team Meetings (Level 10)** | Weekly goal-setting & check-ins |
