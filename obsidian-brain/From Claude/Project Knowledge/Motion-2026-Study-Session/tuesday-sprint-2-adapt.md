## Summary

This Week 7 Tuesday class focuses on the "Exploit" phase of the Sprints curriculum: scaling and adapting winning ad concepts. Sophia Beauvoir (Ready Set) explains how iteration strategy must shift in the post-Andromeda Meta landscape, emphasizing meaningful pivots over minor tweaks. Viti Videtta (Happy Mammoth) details a structured process for generating original ideas (gather → work → eureka → develop) and a "creative scaling engine" for multiplying winners. Janae LeVander (Caraway) provides a comprehensive blueprint for building a large-scale UGC creator program, including detailed briefing templates, customization principles, and common mistakes to avoid.

## Frameworks / models / taxonomies presented

- **Name**: Explore vs. Exploit
  - **Definition**: A model for categorizing creative testing efforts into either finding new winning concepts (Explore) or scaling existing ones (Exploit).
  - **Structure**:
    - **Explore**: Testing/experimenting; no winning angle yet; looking for new personas; unsure how to sell new SKU.
    - **Exploit**: Know what works; double down; turn winning angles into new formats; double down on winning personas; push winning SKUs with new formats/creators/offers.
  - **Introduced at**: 7:09
  - **Mode**: visual
  - **Attribution**: presenter's own (Motion curriculum)

- **Name**: Dara's Three Tiers of Edits
  - **Structure**:
    - Tier 1: Iterations
    - Tier 2: Post Production (incl. AI voiceover)
    - Tier 3: New Creator Content
  - **Introduced at**: 7:58
  - **Mode**: visual
  - **Attribution**: cites Dara (Week 3)

- **Name**: Golden Rules of Roadmapping
  - **Structure**: (1) Centralize ideas, (2) Rank by confidence, (3) Prioritize speed-to-launch, (4) Map big swings, (5) Leave room for creativity without distraction.
  - **Introduced at**: 9:02
  - **Attribution**: cites Dara (Week 3)

- **Name**: How to Rank Evidence (Iterations)
  - **Structure**: 8-item ranking from high to low confidence (pressure-tested past performance → competitor copying).
  - **Introduced at**: 9:12
  - **Attribution**: cites Dara (Week 3)

- **Name**: Levers to Scale Meaningfully (Post-Andromeda)
  - **Structure**: Visual refreshes; Talent refreshes; Mashups of top-performer elements; Persona testing; Pain point testing; Entire body changes; Visual format testing.
  - **Introduced at**: 16:50
  - **Attribution**: Sophia Beauvoir (Ready Set)

- **Name**: Pre vs. Post Andromeda Iteration Process
  - **Structure**: Winning Ad → Iterate (Low Hanging Fruit) → Expand. Pre-Andromeda low-hanging fruit = hook/visual tweaks; post-Andromeda low-hanging fruit = format swaps, length tests, message reorders. Expansion shifts from "higher level tests" to "new personas, new pain points."
  - **Introduced at**: 12:34 / 20:49
  - **Attribution**: Sophia Beauvoir

- **Name**: Ad Families
  - **Definition**: Branches of content built around a base winning ad to prevent fatigue, organized by personas, desire, or awareness level.
  - **Introduced at**: 20:00
  - **Attribution**: Sophia Beauvoir

- **Name**: 4-Step Idea Production Process
  - **Structure**: GATHER → WORK → EUREKA → DEVELOPMENT
  - **Introduced at**: 28:21
  - **Attribution**: cites James Webb Young, "A Technique for Producing Ideas"

- **Name**: Categories of Questions to Generate Ideas
  - **Structure**: Analogy / Opposite Thinking / Emotional Reframing / Specificity / Cultural Layer
  - **Introduced at**: 32:49
  - **Attribution**: Viti Videtta (Happy Mammoth)

- **Name**: Static → Video → High-Production Testing Ladder
  - **Structure**: Test angles with statics first → if winning, move to short video (VO/text overlay) → if still winning, move to testimonial/high-production.
  - **Introduced at**: 38:55
  - **Attribution**: Viti Videtta

- **Name**: Why Is This Ad Working? Deconstruction Framework
  - **Structure**: Hook → Angle → Format → Emotional driver
  - **Introduced at**: 40:07
  - **Attribution**: Viti Videtta

- **Name**: Iteration Levers for Multiplying a Winner
  - **Structure**: Different hooks; same script/different creator; different script/same creator; same structure/different script; same script/different B-roll; same concept/different format. Sweet spot: 3–5 batches.
  - **Introduced at**: 42:02
  - **Attribution**: Viti Videtta

- **Name**: Green/Yellow Cheat Code (Iteration Sizing)
  - **Structure**: Green = winner ad → bigger swings; Yellow = scaled ad → smaller swings.
  - **Introduced at**: 49:43
  - **Attribution**: Viti Videtta

- **Name**: The Creative Scaling Engine
  - **Structure**: Winner → Deconstruct → Multiply → Test → Scale → Fatigue → Refresh (cyclical).
  - **Introduced at**: 53:26
  - **Attribution**: Viti Videtta

- **Name**: Caraway UGC Creator Selection
  - **Structure**: Core Persona + Retainers (reliable baseline) vs. Expansion (testing new/niche personas).
  - **Introduced at**: 58:57
  - **Attribution**: Janae LeVander

- **Name**: Caraway Briefing Blueprint
  - **Structure**: Content Buckets (Evergreen/BAU, Product Launches, Seasonal Pushes) + Research Steps (What worked last year? / What's working in the wild? / What's our hypothesis?).
  - **Introduced at**: 60:28
  - **Attribution**: Janae LeVander

- **Name**: Caraway Concept Template
  - **Structure**: Concept Title / Angle (Why it works) / Creative Guardrails / Objection Handling / Hook Starters (3–5).
  - **Introduced at**: 62:00
  - **Attribution**: Janae LeVander

- **Name**: Caraway UGC Brief Template (9 sections)
  - **Structure**: Project Overview / Filming Guidelines / Deliverables Upload List / UGC Examples / Hooks / CTAs / Example Voiceover / Brand Talking Points / Shot List.
  - **Introduced at**: 63:48
  - **Attribution**: Janae LeVander

- **Name**: Why Generic Briefs Fail
  - **Structure**: Study creator's style → Adapt the concept → Refresh often.
  - **Introduced at**: 67:43
  - **Attribution**: Janae LeVander

- **Name**: 4-Week UGC Program Starter Plan

## Tactics recommended (do this)

- **Tactic**: Deconstruct winners into Hook / Angle / Format / Emotional driver before scaling. — **Speaker**: Viti Videtta — **Timestamp**: 40:07
- **Tactic**: Test angles with statics first, then graduate to short video, then to high-production testimonials. — **Speaker**: Viti Videtta — **Timestamp**: 38:55
- **Tactic**: Run 3–5 batches of iterations per winning concept (different hooks, scripts, creators, B-rolls, formats). — **Speaker**: Viti Videtta — **Timestamp**: 42:02
- **Tactic**: When a winner fatigues, build a "v2" — new script, new hooks, new visuals — but keep the soul/north of the original. — **Speaker**: Viti Videtta — **Timestamp**: 51:17
- **Tactic**: Tag iterations green/yellow in your tracker — green winners get bigger swings, yellow scaled ads get smaller swings. — **Speaker**: Viti Videtta — **Timestamp**: 49:43
- **Tactic**: Use visual refreshes with new footage to scale meaningfully under Andromeda. — **Speaker**: Sophia Beauvoir — **Timestamp**: 16:55
- **Tactic**: Use talent refreshes (single vs. multi-talent) for meaningful variation. — **Speaker**: Sophia Beauvoir — **Timestamp**: 17:01
- **Tactic**: Mash up winning elements from prior top performers (e.g., high-thumbstop ad + high-CVR ad). — **Speaker**: Sophia Beauvoir — **Timestamp**: 18:17
- **Tactic**: Test personas, pain points, and entire ad-body rewrites as scaling levers — not just visuals. — **Speaker**: Sophia Beauvoir — **Timestamp**: 18:43
- **Tactic**: Test visual format swaps (statics ↔ GIFs ↔ video ↔ carousel) on validated messaging. — **Speaker**: Sophia Beauvoir — **Timestamp**: 19:15
- **Tactic**: Build "ad families" — branches of content rooted in a winning concept, organized by persona/desire/awareness level. — **Speaker**: Sophia Beauvoir — **Timestamp**: 20:00
- **Tactic**: Focus intentional effort on the opening 3 seconds of every ad — Andromeda predicts fatigue and limits delivery if hook rate is weak. — **Speaker**: Sophia Beauvoir — **Timestamp**: 22:25
- **Tactic**: Build a footage library based on account learnings (e.g., always request a "couple dancing" shot if that motif compounds). — **Speaker**: Sophia Beauvoir — **Timestamp**: 16:13
- **Tactic**: Gather both specific knowledge (product/brand/customer) and general knowledge (culture, memes, psychology) before ideating. — **Speaker**: Viti Videtta — **Timestamp**: 29:18
- **Tactic**: Force connections between specific and general inputs to produce non-generic headlines. — **Speaker**: Viti Videtta — **Timestamp**: 30:01
- **Tactic**: Use 5 question categories (Analogy / Opposite / Emotional Reframing / Specificity / Cultural) to generate ideas. — **Speaker**: Viti Videtta — **Timestamp**: 32:49
- **Tactic**: Submit ideas to your team for criticism — good ideas have self-expanding qualities. — **Speaker**: Viti Videtta — **Timestamp**: 36:00
- **Tactic**: Customize briefs to each creator's natural style (educational/aesthetic/comedic). — **Speaker**: Janae LeVander — **Timestamp**: 68:18
- **Tactic**: Always read voiceover scripts aloud before sending. — **Speaker**: Janae LeVander — **Timestamp**: 73:05
- **Tactic**: Write briefs in plain language — "show how easy it is to clean the pan," not "demonstrate product value props." — **Speaker**: Janae LeVander — **Timestamp**: 73:19
- **Tactic**: Include hooks at the concept stage (before the brief template) to pressure-test whether the concept is worth pursuing. — **Speaker**: Janae LeVander — **Timestamp**: 63:25
- **Tactic**: Build objection handling directly into the concept page so the brief preempts buyer pushback. — **Speaker**: Janae LeVander — **Timestamp**: 62:38
- **Tactic**: Hold regular syncs with media buyers and CX so briefs respond to live performance gaps. — **Speaker**: Janae LeVander — **Timestamp**: 72:25
- **Tactic**: Specify mandatory deliverables in creator contracts so missing elements trigger reshoots. — **Speaker**: Janae LeVander — **Timestamp**: 74:01
- **Tactic**: For new UGC programs, follow the 4-week starter plan: audit calendar → sync with media buyer → build/test brief → analyze and iterate. — **Speaker**: Janae LeVander — **Timestamp**: 74:05

## Anti-patterns (don't do this)

- **Anti-pattern**: Running 10 minor-text-tweak variants of the same image — Andromeda treats them as one retrieval ticket. — **Speaker**: Sophia Beauvoir — **Timestamp**: 13:30
- **Anti-pattern**: Letting a base winner run on its own without building an ad family around it. — **Speaker**: Sophia Beauvoir — **Timestamp**: 20:14
- **Anti-pattern**: Iterating the same ad over and over until all ads start to look the same. — **Speaker**: Sophia Beauvoir — **Timestamp**: 22:08
- **Anti-pattern**: Copying competitor ads from scratch instead of building original combinations. — **Speaker**: Viti Videtta — **Timestamp**: 26:34
- **Anti-pattern**: Stopping at the first idea instead of generating many combinations. — **Speaker**: Viti Videtta — **Timestamp**: 32:08
- **Anti-pattern**: Forcing every creator into the same top-performing brief template. — **Speaker**: Janae LeVander — **Timestamp**: 68:25
- **Anti-pattern**: Reusing stale brief templates without refreshing hooks/examples. — **Speaker**: Janae LeVander — **Timestamp**: 68:57
- **Anti-pattern**: Siloing the creative team from media buyers. — **Speaker**: Janae LeVander — **Timestamp**: 72:23
- **Anti-pattern**: Saying yes to every leadership idea without strategic rationale. — **Speaker**: Janae LeVander — **Timestamp**: 72:38
- **Anti-pattern**: Sending voiceover scripts without reading them aloud first. — **Speaker**: Janae LeVander — **Timestamp**: 73:05
- **Anti-pattern**: Using marketing jargon in creator-facing briefs. — **Speaker**: Janae LeVander — **Timestamp**: 73:19
- **Anti-pattern**: Vague editor briefs without visual references or style specs. — **Speaker**: Janae LeVander — **Timestamp**: 73:50
- **Anti-pattern**: Weak contracts without reshoot clauses or mandatory-element specs. — **Speaker**: Janae LeVander — **Timestamp**: 74:01
