## Summary

This Week 2 group coaching session focuses on research methodologies for creative strategy. Coaches answer student questions about testing emotionally strong but uncommon micro-moments, finding new entry points beyond a winning comparison ad (Kolkata Chai Co.), and determining when to stop researching and start testing. Eric Philippou presents his "Grocery List Mindset" and frameworks for ideation in extreme account scenarios (nothing working vs. everything working), illustrated with a Flowell case study, while Jake Abrams and Alysha walk through AI-powered research and roadmapping workflows using Claude (Projects, Skills), Whisper, and a custom TikTok scraper.

## Frameworks / models / taxonomies presented

- **The Two Extremes (account standing)** — Eric Philippou (own)
  - Nothing working → best long-term learning accounts; go wide, big swings, net-new concepts
  - Everything working → best short-term financial impact; go deep, maximize, persuade the algorithm
  - Introduced 25:49

- **"Grocery List" Ideation Mindset** — Eric Philippou (own)
  - Do analysis first → know exactly what asset/format you need (e.g., "static with headline + 3 benefits") → then enter swipe files with a list. Ideation should take <10 min.
  - Introduced 23:39

- **Battleship Mindset (when nothing is working)** — Eric Philippou (own)
  - Test many different things; only iterate when you get a "hit" or partial-metric win
  - Introduced 27:55

- **Redefined "Win" Metrics** — Eric Philippou (own)
  - Beyond ROAS/CPA: break down funnel; a dramatic improvement in a single metric (e.g., CPC, hook %, CVR) is a small victory worth iterating on
  - Introduced 28:22

- **Common Symptoms of "Nothing Working"** — Eric Philippou (own)
  - Boring, Complexity, CPM Hell, No Clear UVP — each with its own solution
  - Introduced 34:33

- **Creative Mix for Aggressive Scaling (50/25/25)** — Eric Philippou (own)
  - 50% blatant copies of winners (visually unique, same/similar copy)
  - 25% dramatized winner iterations (more emotional/humorous; "write like Ali Wong")
  - 25% sub-angle tests (deeper niches inside the winning angle)
  - Introduced 43:48

- **Three Signal Sources for Research** — Jake Abrams (own)
  1. Ad account performance
  2. Customer/brand signal (reviews, sales calls, CRM)
  3. External organic signal (TikTok, Reddit)
  - Introduced ~54:28

- **Sarah's framework recap (cross-week)** — Sarah Burnett (Week 2 Tuesday)
  - Generational Insights, Micro-Moments, Emotional Mapping (Valence-Intensity), TEEP (Trigger / Exploration / Evaluation / Purchase)
  - Re-referenced 07:52

## Tactics recommended (do this)

- Do analysis before ideation; enter swipe files with a specific "grocery list" of what you need — Eric, 24:12
- When nothing's working, mine top-performing organic TikToks in the niche for hooks, visuals, language, and audience vibe; replicate hooks frame-for-frame and tie to your product — Eric, 31:02
- Use authority-figure creators (e.g., nurses) for complex/medical products — Eric saw watch time jump from ~9s to 25s+ on the same script — Eric, 39:50
- Prioritize unique-value-prop benefits in reviews over the most common benefit (e.g., "no bloating" over "tastes great") — Eric, 31:32
- When everything's working, allocate 50% blatant visual copies / 25% dramatized / 25% sub-angle tests — Eric, 43:48
- Tell Claude to "write like Ali Wong" for dramatized iterations — Eric, 44:34
- Use disclaimers/asterisks to push claim boundaries while staying compliant — Eric, 50:50
- For research cadence, do a deep dive monthly/quarterly + a 20-min refresh weekly — Jake, 53:47
- Always require AI to cite where each insight came from — Jake, 62:55
- Use Whisper for stream-of-consciousness prompt drafting, then have AI structure the prompt — Jake, 65:21
- Use Claude Projects to store brand-specific memory (review CSVs, briefs); use Claude Skills for repeatable processes/frameworks — Alysha, 67:20
- Reverse-engineer competitor/example ads with an AI prompt that applies Sarah's framework — Alysha, 15:16
- Test a hunch with a low-fidelity format (static, word wall) within 24 hours; reserve high-fidelity formats (street interviews, staged moments) for higher-confidence ideas — Jake, 08:30
- Max out failed-solution and head-to-head comparison angles (and all motivators within them) before moving to other angle types — Jake, 21:35
- Pull the language/slang of the niche from organic TikTok/Google Trends to choose ad copy verbiage (e.g., "semaglutide" vs. "GLP-1") — Eric, 32:30

## Anti-patterns (don't do this)

- Don't ideate before analysis — you'll wander swipe files aimlessly — Eric, 24:20
- Don't keep launching net-new concepts when you have a clear winner — that wastes the scaling window — Eric, 42:37
- Don't rely on minor copy/headline tweaks as "iterations" — Meta's algorithm now treats them as duplicates and deprioritizes spend — Eric, 43:30
- Don't blindly trust AI output without source attribution — Jake, 62:55
- Don't try to out-compete category giants on their turf (e.g., Flowell vs. LeBron-tier athlete sponsorships) — find a blue-ocean persona instead — Eric, 38:18
