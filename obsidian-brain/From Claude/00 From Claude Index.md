# 📦 From Claude — Archive Index

Everything extracted from the **Claude.ai data export** (the `.zip` from Settings →
Privacy → Export data), filtered to Adriana's in-scope areas. This is the **raw
reference layer** that sits underneath the curated brain — the curated notes in
`00 START HERE` distill the durable knowledge; this folder holds the full source.

## What's here
| Folder | What it is |
|---|---|
| **Condensed Memory/** | Claude's auto-condensed memory per in-scope project + account-wide. Dense and current — the single richest source. |
| **Project Knowledge/** | Full knowledge docs Adriana loaded into her Claude projects (Adrianavmarquez Advertising Ideation, Advertising Copywriting, Creatividad², Motion 2026 Study Session). |
| **Chat Archive/In-scope/** | One `.md` per chat classified as clearly in-scope. |
| **Chat Archive/Review/** | Ambiguous / untitled chats — quarantined for you to sort, not auto-filed. |
| **_Extraction Report.md** | Counts + the list of chats excluded for mentioning other ventures. |

## Highest-value reads
- **`Condensed Memory/adrianavmarquez.md`** — positioning, partnerships, rate rules,
  voice lexicon, platform logic, full gear + brand system. *(Feeds most curated notes.)*
- **`Condensed Memory/Consulting-and-Assesments-adrianavmarquez.md`** — the VOC method,
  Cinematic Confessional, POV formula, output preferences.
- **`Condensed Memory/Creatividad2.md`** — the ecosystem, the 16-skill system, series naming.

## Scope & redaction notes
- **Excluded businesses** (Ricos Açaí, Celestina, Sparked Reactions, Bulk HVAC and their
  clients) were walled off: their project docs/memories were skipped, chats mentioning
  them were left out (titles listed in the Extraction Report), and any incidental line
  referencing them was **redacted** → `[line redacted — references an out-of-scope venture]`.
- Conversations in this export carry **no project link**, so chats were routed by
  **content classification**, not by project. Anything the classifier couldn't place
  confidently went to **Review/** rather than into the brain.
