# Creator Maturity Model (6 Buckets)

**One line:** Adriana's proprietary self-diagnostic — *"¿Por qué tu contenido no
convierte?"* — that scores a creator across six stages to find the **bottleneck**.

> [!important] This is signature IP
> It powers the **[[Creatividad2 Course]]** funnel (a free 12-question quiz →
> personalized 5-minute plan). The lowest-scoring bucket = where it all falls apart.

## The six buckets (the value chain)
1. **Estrategia** — do you know *who* it's for and *why* they follow?
2. **Pre-producción** — do you know your hook/message *before* recording?
3. **Producción** — light, audio, coverage (main take + cutaways)?
4. **Edición** — retention-first cutting, subtitles, rhythm?
5. **Distribución** — platform-adapted posting + SEO captions?
6. **Análisis** — reading the right metrics and **making wins repeatable**?

The diagnostic reports a level per bucket (higher = stronger). Lowest bucket shown in
red `#D72323`, the two next-weakest in yellow `#FFBA35`, healthy ones in lavender.

## Two tracks
- **Track A — Creator:** personal brand, *you* are the face.
- **Track B — Business/Product:** the brand isn't you, even if you run it.

## The scoring logic (what "good" looks like)
Each answer maps to 0 (mature) → 3 (immature); there's also a *conchita* (trap/vanity)
option that looks good but isn't. Mature (0-point) answers, by bucket:

| Bucket | The mature move |
|---|---|
| Estrategia | A clear reason to follow **beyond the topic itself**; decide content from a **calendar / customer need**, not trends |
| Pre-producción | Hook & message defined **before** recording; pull from a **banco de ideas** |
| Producción | Prioritize **light + clean audio**; shoot **main take + cutaways** |
| Edición | Edit so it **hooks and never loses anyone**; **subtitles always** |
| Distribución | **Adapt per channel**; caption says something searchable and connects |
| Análisis | Watch **saves / watch time / shares**; understand *why* a winner won and **turn it into a manual** |

The *conchita* traps (avoid): "test it fast because it's trending," pretty
backgrounds over light, animated trendy subtitles for their own sake, one-tool
cross-posting, watching the *full* dashboard instead of the signal metrics.
