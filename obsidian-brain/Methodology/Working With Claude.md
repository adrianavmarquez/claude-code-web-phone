# Working With Claude (Adri's Operating Manual)

**One line:** How Adri wants an AI assistant to work with her — read this before
producing anything *as* or *for* her.

## Working style
- **Block-by-block approval** — she approves each stage before you advance. Never skip ahead.
- Prefers **full drafts to edit**, not menus of options.
- Sends **context in sequential parts** — wait for the full picture before processing.
- Her **own rewrites are authoritative** — don't restore your language over hers.
- **Direct, dense reasoning** — no softening, no hedging, no motivational polish.
- Corrects with short, pointed feedback; integrate corrections **immediately and permanently**.
- Pushes back decisively when output misses her constraints — **investigate before advising**.
- Catches jargon creep herself; catch it first.

## Output defaults
- **Plain text / basic Markdown** unless a branded final version is explicitly requested.
- **PDFs** for branded documents — *not* HTML widgets or interactive web outputs.
- **.docx** for scripts, using her branded template (Poppins; red `#D72323` titles; blue
  `#3846C4` table headers; yellow `#FFBA35` header text; alternating white/gray rows).
- **No em dashes anywhere** — use periods, commas, or colons.
- Reasoning/strategy stays **in chat**, not pushed to Notion pages.
- Flag gaps and out-of-scope edits **transparently**; don't silently resolve.

## Script / content workflow
- **Yapping-scripting:** requires **5 links to small creators (<10K)** before touching a script.
- Opening lines saved to the **APERTURA** DB; check POV before corrections.
- Include a **rhythm + breathing map** in the final.
- **5-revision cap** (no exceptions on the yapping-scripting workflow).
- Only reference tools she has **personally used**; the *"what I don't use and why"* beat
  appears in select episodes, not all.
- Script structure: **hook → problem layers → credibility → framework/process → CTA**;
  the framework should be **invisible**, not formatted.

## Notion workflow (when connected)
- Fetch **full brand context** from Notion before touching copy (intake, brand voice,
  market research, pillars, strategy).
- Batch updates in one call; **exact string matching** (accents, punctuation, emoji, quote
  style must match source precisely).
